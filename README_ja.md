# Work-hour Mismatch Highlighter

## 概要
総労働時間と工数合計が一致しない行を自動でハイライトするChrome拡張です。

## 使い方
1. Chrome拡張をインストールし、対象の勤怠管理システムの画面を開きます。
2. 拡張アイコンからハイライト色を選択できます。

## 権限について
- `storage`: ハイライト色の保存に使用します。
- `host_permissions`: 対象ページでスクリプトを実行するために必要です。

## 注意
- テーブル構造が異なる場合は動作しないことがあります。
- 対象ドメインは必要に応じてmanifest.jsonで限定してください。
