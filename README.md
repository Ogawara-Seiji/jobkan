# 【ジョブカン】総労働時間 ≠ 工数合計 をハイライト

[![Chrome Web Store](https://img.shields.io/chrome-web-store/v/ejfgpbohjkefokdcdnklcohcenmchgpa.svg)](https://chromewebstore.google.com/detail/ejfgpbohjkefokdcdnklcohcenmchgpa)

## 概要
ジョブカン（Jobcan）の勤怠・工数管理画面において、「総労働時間」と「工数合計」が一致していない日を自動的にハイライトし、視覚的に分かりやすくするChrome拡張機能です。

## 特徴
月末の締め作業や、日々の工数入力の確認をスムーズにするための以下の特徴があります。

* **自動ハイライト**
  * ページを開くだけで、時間が合わない箇所がひと目で分かります。
* **確認作業の時短**
  * 入力漏れや計算ミスの発見が容易になり、修正作業の負担を軽減します。
* **シンプルな動作**
  * 複雑な設定は一切不要です。ブラウザにインストールするだけで即座に機能します。

## インストール方法
1. [Chrome Web Store](https://chromewebstore.google.com/detail/ejfgpbohjkefokdcdnklcohcenmchgpa) にアクセスします。
2. 「Chromeに追加」ボタンをクリックしてインストールします。

## 使い方
1. 本拡張機能をインストールします。
2. ジョブカンにログインし、工数管理（または勤怠）の画面を開きます。
3. 「総労働時間」と「工数合計」に差異がある行が、自動的に強調表示されます。
